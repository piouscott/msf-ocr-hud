MSF Counter - Guide d'Installation

Version Tester

Etape 1 : Telecharger l'extension

1 Ouvrez ce lien dans votre navigateur :

https://asperatus.lsjsb.xyz/public.php/dav/files/Sg7q3LyR7tQZSpN/?accept=zip

2 Le fichier ZIP se telecharge automatiquement

3 Extrayez le ZIP dans un dossier de votre choix

  • Clic droit sur le fichier > "Extraire tout..."
  • Choisissez un emplacement (ex: Documents/MSF-Counter)
Important : Gardez ce dossier apres extraction, ne le supprimez pas ! L'extension en a besoin pour fonctionner.

Etape 2 : Installation selon votre navigateur

üåê Google Chrome

1 Ouvrez Chrome et tapez dans la barre d'adresse :

chrome://extensions

2 Activez le "Mode developpeur" (interrupteur en haut a droite)

3 Cliquez sur "Charger l'extension non empaquetee"

4 Selectionnez le dossier extrait (celui contenant manifest.json)

5 L'extension apparait ! Cliquez sur l'icone puzzle puis epinglez "MSF Counter"

ü¶ä Mozilla Firefox

1 Ouvrez Firefox et tapez dans la barre d'adresse :

about:debugging#/runtime/this-firefox

2 Cliquez sur "Charger un module temporaire..."

3 Naviguez jusqu'au dossier extrait et selectionnez le fichier manifest.json

4 L'extension apparait dans la barre d'outils

Note Firefox : L'extension doit etre rechargee a chaque redemarrage du navigateur. Pour une installation permanente, utilisez Firefox Developer Edition.

üéµ Vivaldi

1 Ouvrez Vivaldi et tapez dans la barre d'adresse :

vivaldi://extensions

2 Activez le "Mode developpeur" (interrupteur en haut a droite)

3 Cliquez sur "Charger l'extension non empaquetee"

4 Selectionnez le dossier extrait (celui contenant manifest.json)

5 L'extension est installee !

Etape 3 : Utilisation

1 Cliquez sur l'icone de l'extension dans la barre d'outils

2 Cliquez sur le bouton "Counters"

3 Parcourez les equipes et leurs counters recommandes

4 Utilisez FR / EN pour changer la langue

Depannage

Probleme Solution

Extension disparue (Firefox) Rechargez via about:debugging
Pas d'icone dans la barre Cliquez sur l'icone puzzle (Chrome) et epinglez
Portraits non affiches Verifiez votre connexion internet
